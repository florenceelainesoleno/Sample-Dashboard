# 🌍 Drivers of Life Expectancy Dashboard

> **What truly helps people live longer — wealth, education, or healthcare?**

Explore this interactive dashboard and uncover how key development indicators shape life expectancy across different countries and years.

---

## 📌 Project Overview

The **Drivers of Life Expectancy Dashboard** is an interactive data visualization project built using **Python** and **Streamlit**. It analyzes how different socioeconomic factors such as:

- 💰 GDP per Capita  
- 🎓 Education Index  
- 🏥 Health Index  

influence **Life Expectancy** around the world.

This dashboard transforms raw data into meaningful visual insights through charts, KPI cards, filters, and comparisons.

---

## 🎯 Objectives

✔ Analyze trends in life expectancy over time  
✔ Compare countries based on development indicators  
✔ Discover relationships between income, education, and health outcomes  
✔ Present findings through an engaging interactive dashboard  

---

## 🚀 Features

### 📊 KPI Cards
Instantly view average values for:

- Life Expectancy  
- GDP per Capita  
- Education Index  
- Health Index  

### 🌎 Country Comparison
Compare selected countries in a chosen year.

### 📈 Trend Analysis
Track the global life expectancy trend over time.

### 🔍 Relationship Analysis
See how GDP relates to Life Expectancy through scatter plots.

### 🫧 Multi-variable Bubble Chart
Explore GDP, Education, and Life Expectancy in one chart.

### 💡 Research-Based Insights
Includes supported findings from economic and public health literature.
